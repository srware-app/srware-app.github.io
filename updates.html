How to check your SRWare Iron version

From within the browser

  1. Open menu → Help → About SRWare Iron (or type iron://settings/help).
  2. The About page shows the current version and whether updates are installed.
  3. If automatic updates are available, Iron will usually check and apply them depending on the build.

Manually check the installer

Compare your installed version to the vendor’s latest release notes or the installer’s metadata. We provide a direct link to the official 64‑bit installer.

Safely update SRWare Iron

  • Always download installers from the official domain (srware.net) or vendor pages.
  • Keep a backup of your profile (bookmarks and important settings) before major updates.
  • When in doubt, perform a clean reinstall (see Troubleshoot → Clean reinstall).
  • Keep Windows updates and drivers current for best compatibility.

Auto‑update behavior & tips

Auto‑update considerations

  • Some vendor builds auto‑update; others require manual download and install.
  • Portable builds typically require manual replacement for updates.
  • Enterprises may manage updates centrally; consult your admin if in a managed environment.

Recommended update routine

  • Enable automatic updates if offered and you trust the source.
  • Periodically check this page or the vendor’s release notes for major changes.
  • Keep backups and follow a rollback plan in case of incompatibilities.

Updates FAQ

How often are updates released?
Frequency varies: security patches follow upstream Chromium releases, while vendor feature updates depend on the maintainer’s cadence.
Where can I find official release notes?
Check the vendor site or official SRWare/GitHub pages. Use the direct download link we provide for installer access.
Can I roll back an update?
Rolling back typically requires reinstalling a previous version and restoring a backup of your profile. Keep versioned backups if rollback is important.
