Troubleshooting SRWare Iron on Windows

Fix common issues: won’t start, crashes or freezes, installation blocked, slow performance, and video problems.

Download installer (64‑bit) Install Guide Clean reinstall

Works on Windows 11/10 (64‑bit). For locked‑down PCs, consider the portable version.

Common issues & quick fixes

Iron won’t start

  • End stuck processes (Task Manager → end “iron.exe”).
  • Check antivirus/quarantine and restore if needed.
  • Run as administrator once to rule out permission issues.
  • Create a fresh profile: rename the Iron user profile folder and relaunch.

Crashes or freezes

  • Disable or remove recent extensions.
  • Toggle hardware acceleration (Settings → System).
  • Update GPU drivers and Windows updates.
  • Reset settings or try a clean profile.

Can’t install

  • SmartScreen: click “More info” → “Run anyway” if you trust the source.
  • UAC blocked: use an admin account or portable build.
  • Offline? Re‑download with a stable connection.
  • Path issues: install to a user‑writable folder.

Slow performance or video issues

Speed up browsing

  • Clear cache and cookies.
  • Disable heavy extensions and background tasks.
  • Close tabs with live streams or auto‑refresh.

Hardware acceleration

Toggle hardware acceleration: some GPUs benefit, others cause stutters.

Settings → System → “Use hardware acceleration when available”.

Video codecs

Ensure Windows media components are updated. Try another extension for playback or check site‑specific settings.

Clean reinstall (step‑by‑step)

  1. Backup important data

    Export bookmarks and note passwords. If using a portable profile, copy the entire folder.

  2. Uninstall SRWare Iron

    Windows Settings → Apps → Installed apps → SRWare Iron → Uninstall.

  3. Remove leftover profile

    Delete or rename the Iron user profile folder to avoid conflicts when reinstalling.

  4. Reinstall from the official link

    Download again from the official server and run the installer.

    Download installer

  5. Start fresh and test

    Launch Iron and test basic sites before restoring extensions or imports.

If issues persist after a clean reinstall, check Updates and ensure your Windows drivers and system updates are current.

Troubleshooting FAQ

Where is the Iron profile folder?
Profile locations depend on the build and setup. If using a portable build, your profile is inside the portable folder. Otherwise search for “Iron” in your AppData or installation directory.
SmartScreen blocks the installer — is it safe?
Use the official link and verify the source (srware.net). Proceed only if you trust the origin. Keep antivirus enabled.
Extensions cause crashes — what now?
Disable extensions in Safe Mode‑like fashion (start with “--disable-extensions”) or remove recent ones, then re‑enable selectively.
