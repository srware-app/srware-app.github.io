Install SRWare Iron on Windows 11/10

Get Iron up and running in minutes. Follow these steps to install, set it as default, and import your data.

Download installer (64‑bit) Troubleshoot install

Works on Windows 11/10 (64‑bit). If you can’t install, consider the portable version.

Step‑by‑step installation

  1. Download the installer

    Click “Download installer (64‑bit)” above. The file srware_iron64.exe comes directly from srware.net.

    Keep your antivirus enabled. Optional: verify the file hash if checksums are provided by the vendor.

  2. Run the installer

    Double‑click the downloaded EXE. If Windows SmartScreen appears, click “More info” → “Run anyway” only if you trust the source.

  3. Approve UAC (if prompted)

    Windows may ask for permission to make changes. Click “Yes” to continue.

  4. Choose install options

    Select the destination folder, create a desktop shortcut if desired, and proceed with the default options unless you have a specific need.

  5. Launch SRWare Iron

    After setup completes, open Iron. On first run, you can import bookmarks and passwords from another browser.

  6. Set as default (optional)

    Windows Settings → Apps → Default apps → search for “Iron” → set it as the default for HTTP/HTTPS and .html links.

  7. Harden privacy (recommended)

    Visit Privacy Setup for recommended settings and extensions (e.g., content blockers, HTTPS enforcement).

Tips and maintenance

Update Iron

Check for new releases regularly. See Updates for version info and tips.

Import/Export data

Menu → Bookmarks → Import Bookmarks and Settings. You can also export bookmarks for backup.

Uninstall or reset

Use Windows Settings → Apps to uninstall. For a clean start, remove or rename the user profile folder before reinstalling.

Install FAQ

Do I need admin rights to install?
In most cases no, but certain environments may require admin approval. Try the portable version if you cannot install.
Is this guide for Windows 11 and 10?
Yes. The steps are the same for Windows 11 and Windows 10 (64‑bit).
How do I set Iron as default?
Windows Settings → Apps → Default apps → choose Iron and set it for HTTP/HTTPS and .html files.
